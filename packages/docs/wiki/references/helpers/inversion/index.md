# Inversion Helper

Core DI system enabling loosely coupled, testable, and extensible code. Standalone IoC container with decorator-based injection, fluent binding API, and singleton/transient scoping.

> [!NOTE]
> The standalone DI container lives in `@venizia/ignis-inversion`. The framework package `@venizia/ignis` re-exports everything and adds higher-level helpers (`app.controller()`, `app.service()`, etc.).
